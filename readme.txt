Detect and Block spam comments.
== Description ==

Main feature and goal of this plugin 'Block Spam Comments' is to detect spam comments and create a shield to protect from submit this unwanted robot comments without using captcha. Plugin size is less than 1 KB, so no tension about storage size. Install and Activate then enjoy spam free website.


== Installation ==

Installation is very easy like other wordpress plugin
e.g.

1. Upload `Block Spam Comments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


= How do I install this plugin? =
You can install as others regular wordpress plugin. No different way. Please see on installation tab.

= Any further process to use ? =
Nothing else after activating, lets enjoy.
